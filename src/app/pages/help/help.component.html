Manual de Usuario

1. Introducción

Este manual de usuario tiene como objetivo proporcionar instrucciones claras para utilizar el sistema de filtrado web, permitiendo la gestión de reglas, usuarios y accesos de manera intuitiva. El sistema está diseñado para interceptar tráfico web, aplicar reglas configuradas y registrar acciones realizadas.

2. Requisitos Previos

  • Acceso a un navegador compatible (Google Chrome, Mozilla Firefox, etc.).
  • Conexión al sistema mediante la URL del frontend proporcionada.
  • Credenciales válidas (usuario y contraseña) para acceder al sistema.

3. Inicio de Sesión

  • Acceda al sistema mediante la URL proporcionada.
  • Introduzca sus credenciales en el formulario de inicio de sesión.
  • Haga clic en el botón "Iniciar sesión".

4. Funcionalidades Principales

4.1 Dashboard

  • Acceso: Una vez iniciada la sesión, el sistema le llevará al dashboard principal.
  • Funciones disponibles:
    • Visualizar gráficos de actividad: páginas bloqueadas, URLs accedidas, etc.
    • Consultar el estado del proxy (activo/inactivo).

4.2 Gestión de Reglas

  • Acceda al menú "Reglas" en la barra de navegación.
  • Opciones disponibles:
    • Crear Regla: Complete el formulario con URL, IPs y roles, y haga clic en "Guardar".
    • Editar Regla: Seleccione una regla, modifíquela y haga clic en "Actualizar".
    • Eliminar Regla: Seleccione una regla y haga clic en "Eliminar".

4.3 Historial de Navegación

  • Acceda al menú "Historial".
  • Funcionalidades:
    • Filtrar registros por fecha, usuario, IP, URL o acción.
    • Exportar el historial en formatos compatibles (CSV, PDF).

4.4 Gestión de Usuarios

  • Acceda al menú "Usuarios".
  • Funciones disponibles:
    • Añadir Usuario: Complete el formulario con los datos y roles del usuario.
    • Editar Usuario: Seleccione un usuario, modifíquelo y haga clic en "Actualizar".
    • Eliminar Usuario: Seleccione un usuario y haga clic en "Eliminar".

4.5 Configuración del Sistema

  • Acceda al menú "Configuración".
  • Funcionalidades:
    • Iniciar/Detener el proxy.
    • Configurar mensajes de bloqueo personalizados.
    • Cargar certificados para conexiones HTTPS.

5. Solución de Problemas Comunes

  • No se puede acceder al sistema: Verifique la URL y la conexión a Internet.
  • Regla no aplicada: Asegúrese de haber recargado el proxy.
  • Historial vacío: Compruebe que existen registros de accesos bloqueados o permitidos.
